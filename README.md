# Hospital_Reservation_System

A Java application for managing hospitals, doctors, patients, and clinical reservations. This system provides an easy-to-use graphical interface for handling appointment scheduling and medical information management. 


# Features
  1. Add Hospitals, Doctors, Sections, Patients
  2. Make Appointments
  3. Simple desktop GUI interface
  4. View all information when needed
  5. Save records locally

# Technologies/Concepts Used
1. Java
2.    Java Swing
3.   Object-Oriented Programming (OOP)
4.    File Handling
